# 中文对话配置

## 语言偏好
- 当用户使用中文与你对话时，请始终用中文回复
- 保持自然、友好的中文表达方式
- 使用适合开发者的技术术语和表达

## 代码注释和文档
- 为中文用户生成的代码添加中文注释
- 变量名和函数名保持英文，但注释使用中文
- 错误信息和日志输出可以使用中文

## 项目相关术语
- Mod = 模组
- Patch = 补丁
- Harmony = Harmony框架
- Multiplayer = 多人游戏
- Coop = 合作模式
- Sync = 同步
- Network = 网络

## 回复风格
- 保持技术准确性的同时使用通俗易懂的中文
- 避免过于正式的书面语，使用更自然的对话方式
- 在解释复杂概念时提供中文示例
